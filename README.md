# 🇳🇬 Nigeria GDP Growth Dashboard

An interactive data analysis dashboard visualizing Nigeria's GDP Growth Rate
from 1961 to 2024, built using Python, Flask, and Chart.js.

## 📊 Features

- GDP explanation and formula for general audiences
- Summary statistics (average growth, best/worst years, recession count)
- Interactive line chart showing 64 years of GDP trend
- Decade-by-decade bar chart comparison
- Auto-generated key insights from real data
- Policy recommendations based on analysis

## 🛠️ Tech Stack

- **Python** — Data processing
- **Pandas** — Data cleaning and analysis
- **Flask** — Web framework
- **Chart.js** — Interactive charts
- **HTML/CSS** — Frontend dashboard

## 📁 Data Source

World Bank Open Data — Nigeria GDP Growth (Annual %)
https://data.worldbank.org/country/nigeria

## 🚀 How to Run Locally

1. Clone the repository
2. Install dependencies:
pip install -r requirements.txt
3. Run the app:
python app.py
4. Open browser at `http://localhost:5000`
